Step 1: Getting to know GoogleBlockly



1. Look through the categories of the blocks, think about what some of the blocks can do

2. Try to drag some blocks from the sidebar into the center and into each other

3. Drag a block to the bottom right corner into the bin. This is how you can delete blocks

Now you can work with GoogleBlockly! Go on to the first step

Step 2: Setting up the connection between the Arduino and the program.


1. This step helps your program communicate with the joystick.

2. Find a block that helps you connect the program to the joystick.

3. Use that block to establish the connection between the program and the joystick.

Step 3: Reading the input from the joystick.


1. This step helps your program understand the movements of the joystick.

2. Find a block that allows your program to receive information from the joystick.

3. Use that block to read the input from the joystick.

Step 4: Creating logical operations using if statements.


1. Think about how the movements of the joystick should affect the player.

2. Find blocks that help you create rules based on the joystick's movements.

3. Connect those blocks to the actions you want the player to perform.

4. Experiment with different combinations of blocks to see what works.
